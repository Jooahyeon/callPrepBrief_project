# 브리프 보관함 (briefs)

`call-prep-brief-generator` 에이전트가 생성한 **콜 프렙 브리프 결과물**을 미팅별로 저장·축적하는 폴더입니다.

## 저장 규칙

- **파일명**: `브리프_<고객사>_<YYYYMMDD>.md`
  - 예: `브리프_ABC전자_20260616.md`
  - 같은 고객사·같은 날짜에 여러 미팅이 있으면 뒤에 `_2`, `_3` 등을 붙입니다. (예: `브리프_ABC전자_20260616_2.md`)
- **포맷**: `_TEMPLATE.md`의 고정 포맷을 그대로 따릅니다. 헤더·구분선(`====`)·번호·불릿을 변경하지 않습니다.
- **출처 링크 보존**: `[ 출처 / 참고 링크 ]` 섹션의 소스 URL은 리서치 근거이므로 **반드시 그대로 남깁니다.** 임의 삭제 금지.

## 생성 방법

1. `call-prep-orchestrator` 실행 → 입력 수집 → 웹 리서치 → 브리프 생성
2. 브리프가 화면에 출력되며, 동일 내용이 이 폴더에 위 파일명 규칙으로 **자동 저장**됩니다.
3. 저장 경로가 안내되면 필요 시 직접 열어 수정할 수 있습니다.
